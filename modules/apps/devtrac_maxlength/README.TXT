INTRODUCTION
------------
Devtrac Maxlength is a Drupal 7 module that adds the capability for users to
set the maximum length for fields in Devtrac.

Specifically the Follow-UP Task (Action Item), Narrative (Field trip),
Public Summary (Field trip), Notes (Field Trip Report Item),
Report  (Field Trip Report Item) and Description (Location) fields in devtrac.
