INTRODUCTION
------------
Devtrac Mapbox is a Drupal 7 module that adds a mapbox settings form to Devtrac
so that users can add their own mapbox layers to Devtrac.

For more information about Devtrac_Mapbox, visit http://www.devtrac.org/node/71
